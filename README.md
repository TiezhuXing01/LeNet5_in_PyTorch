# LeNet5_in_PyTorch
这是一个用 LeNet5 实现手写数字识别的项目。数据集用的是MNIST。

---
## 关于LeNet5.py
该文件包含了 LeNet5 部分、训练部分和测试部分。
直接运行该文件即可得到最终的测试结果。大约为99%.
